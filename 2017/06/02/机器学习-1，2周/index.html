​ 近两周发奋学习Machine Learning的入门课程，因为在看mit 6..824的课程时不时很懂。在地铁上和家里的学习效率很差，学过的东西忘得很快。只有写下来，多复习一下。

Cost Function

Cost function is used to measure the accuracy of our hypothess function. It takes an averate difference of all the results of the hypothesis with inputs from x’s and the actual output y’s
$$
J(\theta_0,\theta1) = \frac {1}{2m} \sum{i=1}^{n}(h_\theta(x_i) - y_i)^2
$$

Gradient Descent

Gradient descent is used to estimate the parameters in the hypothesis function. We are graphing the cost function as a function of the parameter estimates.We are not graphing x and y itself, but hte parameter range of our hypothesis function and the cost resulting from selectiong a particular set of parameters.We put θ0 on the x axis and θ1 on the y axis, with the cost function on the vertical z axis. The points on our graph will be the result of the cost function using our hypothesis with those specific theta parameters. The graph below depicts such a setup. We will know that we have succeeded when our cost function is at the very bottom of the pits in our graph, i.e. when its value is the minimum. The red arrows show the minimum points in the graph.

The way we do this is by taking the derivative of our cost function.The slop of the tangent is the derivative at that point and it will give us a direction to move towards. We make steps down the cost function in the direction with steepest descent. The size of each step is determined by the α, which is called the learning rate.

The gradient descent algorithm is:

repeat until convergence:
$$
\theta_j := \theta_j - \alpha \frac{\partial}{\partial \theta_j}J(\theta_0, \theta_1)
$$

where j = 0,1 represent the feature index number.

*At each iteration j, one should simultaneously update the parameters $\theta_0, \theta_1, \theta_2 … \theta_n$.
