当初的愿望实现了吗？

发表于 Mar 19 2017   |   分类于 Life   |     |   阅读次数

​ 距离上一篇博文，已经将近半年了，这半年完成了买房的大难题，虽然到现在也没入住，不过总算是不用再租房子住了。在这半年里下班回家后就很少看书了，甚至发展到不看书了。编程能力逐渐遗忘，算法能力也都丢光，享受能力倒是大有发展。

​ 今天挨了领导的批评，原因是工作月报没有及时提交，其实已经被领导点名2次了。可能本身就是一个懒散无纪律的人吧。虽然这月报就是走形式，但是我不走影响别人走，下次还是上心吧，或者想想办法用脚本写一个。近来越来越觉得工作没意思，但又有好多工作感觉没处理，每次写周报，开例会总觉得有没做完，没做好的活，但是开完会后一切照旧，下次开会又是一个循环。从下周起一定要把工作安排好，腾出时间来提高自己的技能，学习感兴趣有用的姿势。

​ 今天写这篇文章是为了告诉自己，不要忘记当初的理想：做一个牛逼的人。至于怎么才算牛逼，目前来说就是比身边的同事强，能大部分人能赚到更多地钱。记得大学毕业的时候后悔大学没好好利用时间提高自己。研究生毕业后，又有过类似的懊悔。现在，时间以越来越快的速度了，自己能做的只有抓紧所剩无几的青春，不要在以后抱怨了。看到朋友圈中的同学发的不忘初心。我也送自己两句歌词：当初的愿望实现了吗？事到如今只好祭奠吗？
