Unique Binary Search Trees

发表于 Jul 16 2016   |     |   阅读次数

Given n, how many structurally unique BST’s (binary search trees) that store values 1…n?

For example,
Given n = 3, there are a total of 5 unique BST’s.

思路

  1. 设 n 个节点的有 G（n）种不同的 BST，F(i, n) 是以第 i 个节点为根节点的不同的 BST 的数目;

  2. G(n) = F(1, n) + F(2, n) + ··· + F(n, n);

  3. 分析 F(i, n)，以 i 为根节点（根节点值为 i ），则其左子树中有 i-1 个节点， 右子树中有 n-i 个节点，分别为 G(i-1) 和 G(n-i), 左子树所有的变化 右子树所有的变化，即为以 i 为根节点的 BST 的数目：可知 F(i, n) = G(i-1) G(n-i);

  4. 由以上可知： G(n) = G(0) G(n-1) + G(1) G(n-2) + ··· + G(n-1)*G(0); 且 G(0) = G(1) = 1;

Unique Binary Search Trees II

Given an integer n, generate all structurally unique BST’s (binary search trees) that store values 1…n.
思路

由上题可知，可从 n-1 个节点的 BST 种类得出 n 个节点的 BST 种类，分两种情况讨论：

  1. 第 n 个 节点是根节点，则只需将第 n 个节点的 left 指向 n-1 个节点的 BST 的根节点；

  2. 第 n 个 节点不是根节点则需要遍历 n-1 个节点的右子树，每当右子树中的 right 不为空，则将 n 节点的 left 指向 该节点的 right， 并将该节点的 right 指向 n 节点。当右子树中某节点 right 指向为 nullptr 则将其指向 n；
