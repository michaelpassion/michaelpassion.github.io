Sum of Two Integers

发表于 Jul 20 2016   |   分类于 leetcode   |     |   阅读次数

Calculate the sum of two integers, but you are not allowed to use the operator + and -.

Example:

Given [1, 2], return 3.

思路
先上代码，照代码说；

首先，用 a & b 算出 a 和 b 二进制位均为 1 的值，记为 carry（进位），二进制中 1+1 = 10；
第二步， a = a ^ b, 将 a 和 b 二进制位中不同的位全赋在 a 上，相当于将 a、b中除了同时为零的位和同时为1的位相加并赋值给 a。
第三步，因为 carry 中为 1的位记录的是a、b中同时为 1 的位，将 carry 左移一位，才是 a + b 真正的进位，b = carry << 1。 重复以上过程，直到进位 carry 为 0，最后得出答案。另 carry = a & b = 0 时 a = a ^ b 等价于 a = a + b；
